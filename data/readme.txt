Data ReadMe
 - A collection of raw data used indirectly to support the main application.


Logs
 - Log files from passed LAN parties including raw and processed versions.

Mockups
 - Source files for user interface mockups.

Screenshots
 - Overhead screenshots used to generate tiles.

Scripts
 - Game scripts for CoD, CoD:UO, and AWE.

Vehicles
 - Core game properties files for vehicles.

Weapons
 - Core game properties files for weapons.
