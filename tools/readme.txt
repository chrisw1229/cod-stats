Tools ReadMe
 - A collection of classes used indirectly to support the main application.
 - Use the Eclipse launch configurations in the run folder to execute.


PropertiesReader
 - Used once to convert weapons configuration files into a more readable format.
 - Default format is a single line of all key values in random order separated by \.
 - New format is a sorted list of one key/value per line.

TileCutter
 - Converts a series of master images into a set of tiles.
